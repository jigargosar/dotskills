# Skill dev workflow

## Edit original skill in place
1. Iterate: back up, edit, restart the session, test.
2. Backup → docs/archive/<original-name>-SKILL-v<N>.md

## Edit a copy of original skill
1. Copy original skill folder to/as a sibling 
2. give it a new name and set name: to match.
3. Iterate: edit, back up, restart the session, test.
4. backup → docs/archive/<copy-name>-SKILL-v<N>.md
5. Then update original skill, with upgraded copy. 
6. Backup original, and then delete copy skill
